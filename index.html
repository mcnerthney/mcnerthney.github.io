App Link Demo for BM App

https://mcnerthney.github.io/link is a link to the app
https://mcnerthney.github.io/link/rewards is a reward link to the app
https://mcnerthney.github.io/link/transaction/12431234 is a transactions link to the app

Notes: this is a demo of how app links work

'mcnerthney.github.io' will be replaced with 'secure.greendot.com'

Entering the URL in android chrome browser shows the web site page.

Clicking the URL in a desktop/laptop browser shows the web site page.

Clicking the URL in a phone browser shows the web site page if the app is not instaalled.

Clicking the URL in a phone browser shows the app if it is instaalled.
